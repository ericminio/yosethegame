Update your server to make it accept a GET request with a positive integer parameter and make it respond a JSON containing the prime factors decomposition of this number

For example, receiving

http://your-server/primeFactors?number=300

You should respond

{
    "number" : 300,
    "decomposition" : [ 2, 2, 3, 5, 5 ]
}

Your server will be called with a random number.

In number theory, the prime factors of a positive integer are the prime numbers that divide that integer exactly. Learn more about prime factors decomposition on Wikipedia
You can find several examples of the Prime factors kata on the Internet.
