Update your server and make it accept a GET request on / and respond an html page with a link to your source repository. When accessing your repository, one should read a welcome message referencing this game.

Receiving

http://your-server/

You should respond

an html page containing an element a#repository-link targeting a page whose element #readme contains the text 'YoseTheGame'. The link should be an absolute link.
github and bitbucket both display, in a #readme element, the content of a readme file present in the root folder of a repository.
