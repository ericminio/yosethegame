Time to go further! The world needs your help: a fire began you have to extinguish. Of course, you need some water (W), and as it's a huge fire (F), let's fly a fire-fighting plane (P).

When you receive

http://your-server/fire/geek?width=3&map=...P...WF

You should respond the following Json

{
    map: [
        "...",
	"P..",
	".WF"
    ],
    moves: [
        { dx: 0, dy: 1 },
        { dx: 1, dy: 0 },
        { dx: 1, dy: 0 },
    ]
}

OR

{
    map: [
        "...",
	"P..",
	".WF"
    ],
    moves: [
        { dx: 1, dy: 0 },
        { dx: 0, dy: 1 },
        { dx: 1, dy: 0 },
    ]
}

OR

Any moves that will lead your plane (P) to take water (W) and then reach the fire (F)
