A new fire is calling and you must extinguish it as soon as possible and therefore choose the point of water that minimizes the total length of the flight.

For example, when you receive

http://your-server/fire/geek?width=5&map=W...PF........W

You should respond the following Json

{
    map: [
        "W...P",
	"F....",
	"....W"
    ],
    moves: [
        { dx: -1, dy: 0 },
        { dx: -1, dy: 0 },
        { dx: -1, dy: 0 },
        { dx: -1, dy: 0 },
        { dx:  0, dy: 1 },
    ]
}

We will send a random map and check that your plane takes the shortest way to get water and extinguish the fire.
